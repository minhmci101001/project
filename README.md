# YouTube Trending Predictor — AI

A Machine Learning project that predicts the probability of a YouTube video reaching the **Trending** list **before it is published**, using Random Forest and a Logistic Regression baseline for comparison.

## ✨ Key Features

- **Data Crawling** — Automatically collects trending and non-trending video data via the YouTube Data API v3, including channel subscriber counts.
- **Exploratory Data Analysis (EDA)** — Generates 7 visualizations covering class distribution, view count distribution, correlation heatmap, title length, engagement rate, category breakdown, and publish-hour patterns.
- **Model Training & Evaluation** — Trains a Random Forest classifier (with class balancing) and benchmarks it against a Logistic Regression baseline. Produces 4 model evaluation plots: Confusion Matrix, ROC Curve, Feature Importance, and Model Comparison.
- **Data Leakage Prevention** — Engagement rate (likes/comments/views) is intentionally excluded from model features as these metrics are only available after publication.
- **Modern Web Interface** — A Glassmorphism-styled Flask app that allows real-time predictions with NLP title analysis and interactive feature toggles.

## 🚀 Getting Started

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Configure API Key

Create a `.env` file in the project root:

```
YOUTUBE_API_KEY=Your_API_Key_Here
```

### 3. Crawl data

```bash
python scripts/crawl_data.py
```

This fetches trending and non-trending videos from the YouTube Data API and saves the dataset to `data/youtube_data.csv`.

### 4. Run EDA

```bash
python scripts/eda.py
```

Generates 7 exploratory charts saved to the `plots/` directory.

### 5. Train the model

```bash
python scripts/model_training.py
```

Trains the Random Forest model and the Logistic Regression baseline, prints evaluation metrics, generates 4 model evaluation charts (plots 08–11), and saves `models/trending_rf_model.joblib` and `models/scaler.joblib`.

### 6. Launch the web app

```bash
python app/app.py
```

Open your browser at `http://127.0.0.1:8000/`

## 🤖 Model Features

All features are **pre-publication only** (no data leakage):

| Feature | Description |
|---|---|
| `duration_seconds` | Video length in seconds |
| `title_length` | Number of characters in the title |
| `has_emoji` | Whether the title contains an emoji |
| `is_question` | Whether the title contains a `?` |
| `is_exclamation` | Whether the title contains a `!` |
| `has_number` | Whether the title contains a digit |
| `tags_count` | Number of tags attached to the video |
| `category_id` | YouTube content category |
| `log_subscriber_count` | Log-transformed channel subscriber count |
| `publish_hour` | Hour of day the video was published (UTC) |
| `day_of_week` | Day of the week (0 = Monday, 6 = Sunday) |
